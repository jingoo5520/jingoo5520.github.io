```html
    <head>
        <meta charset="UTF-8" />

        <title>Document</title>
    </head>
    <body></body>
</html>
```

## 테이블 관련 태그

&lt;tr&gt; 태그는 한 행을, &lt;td&gt; 태그는 하나의 칸을 생성한다.<br>
&lt;tr&gt; 태그 대신 &lt;thead&gt; 태그와 &lt;tbody&gt; 태그를 이용해 시맨틱하게 작성도 가능하다.<br>
각 &lt;td&gt; 태그는 &lt;th&gt; 태그를 이용해 직접 헤더임을 표시할 수도 있다.<br>

`<table>`<br>
테이블 구조 생성<br>
`<tr>`<br>
테이블 행 생성<br>
`<th>`<br>
테이블 헤더 칸 생성<br>
`<td>`<br>
테이블 칸 생성<br>
`<thead>`<br>
테이블 행 생성(시맨틱)<br>
`<tbody>`<br>
테이블 행 생성(시맨틱)<br>

> 입력

```html
<table>
    <tr>
        <th>이름</th>
        <th>나이</th>
        <th>전화번호</th>
        <th>성별</th>
    </tr>
    <tr>
        <td>홍길동</td>
        <td>25</td>
        <td>010-0000-0000</td>
        <td>남</td>
    </tr>
</table>
```

```html
<table>
    <thead>
        <th>이름</th>
        <th>나이</th>
        <th>전화번호</th>
        <th>성별</th>
    </thead>
    <tbody>
        <td>홍길동</td>
        <td>25</td>
        <td>010-0000-0000</td>
        <td>남</td>
    </tbody>
</table>
```

> 출력

<table>
    <tr>
        <th>이름</th>
        <th>나이</th>
        <th>전화번호</th>
        <th>성별</th>
    </tr>
    <tr>
        <td>홍길동</td>
        <td>25</td>
        <td>010-0000-0000</td>
        <td>남</td>
    </tr>
</table>

<br>

> &lt;td&gt; 태그의 속성

`colspan` n개의 열을 병합<br>
`rowspan` n개의 행을 병합

> 입력

```html
<table>
    <tbody>
        <tr>
            <td colspan="2">1</td>
            <td>3</td>
        </tr>
        <tr>
            <td>4</td>
            <td>5</td>
            <td rowspan="2">6</td>
        </tr>
        <tr>
            <td>7</td>
            <td>8</td>
        </tr>
    </tbody>
</table>
```

> 출력

<table>
    <tbody>
        <tr>
            <td colspan="2">1</td>
            <td>3</td>
        </tr>
        <tr>
            <td>4</td>
            <td>5</td>
            <td rowspan="2">6</td>
        </tr>
        <tr>
            <td>7</td>
            <td>8</td>
        </tr>
    </tbody>
</table>

<br><br>

# HTML

## HTML 문서 기본 양식

vsCode에서 작성시 ! + [TAB] 을 이용해 기본 양식 자동 완성

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
    </head>
    <body></body>
</html>
```

## 태그와 콘텐트, 그리고 속성

### 제목 태그

`<h1>` `<h2>` ... `<h6>`

> 입력

```html
<h1>제목</h1>
<h2>제목</h2>
<h3>제목</h3>
<h4>제목</h4>
<h5>제목</h5>
<h6>제목</h6>
```

> 출력

# h1 제목

## h1 제목

### h1 제목

#### h1 제목

##### h1 제목

###### h1 제목

### 링크 태그

`<a></a>`

> 입력

```html
<a href="https://www.naver.com">네이버</a>
```

> 출력

<a href="https://www.naver.com">네이버</a>

> 속성

`href` 이동할 링크의 주소나 위치 값
